# 開発者モード  
ライブラリや開発ボードを適応させている場合は、開発者モードをオンにすることを強くお勧めします。

## 開発者モードを開く
`メニュー > 設定 > 開発者モード`を通じて、開発者モードを開きます。
このとき、blocklyワークスペースの左下に`開発者フローティングメニュー`が表示されます。

![](../img/developer/dev-tool.webp)


## 開発者機能
現在提供されている開発者機能は以下のとおりです：  
1. プロジェクトのリロード。  
  クリックするとプロジェクト全体をリロードできます。開発者がライブラリのblockly部分を変更した場合、クリックすると変更されたライブラリをリロードできます；  
2. キャッシュのクリア。  
  クリックするとコンパイルキャッシュをクリアできます。開発者がライブラリsrcソースコードを変更し、以前にコンパイルしたことがある場合、システムが以前のコンパイルキャッシュを使用しないようにキャッシュをクリアする必要があります； 
3. グローバルリソースフォルダを開く
  クリックすると、システムファイルマネージャーを通じて開きます
4. ビルドフォルダを開く

5. 自動保存
  デフォルトでチェックされています。チェックすると、プロジェクトをリロードする前にプロジェクトを保存します。
