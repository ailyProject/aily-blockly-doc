# 基本使用ガイド  

## ガイドページの機能
![](../img/main/main01.png)

①`新規プロジェクト`:新しいプロジェクトを作成します

②`プロジェクトを開く`：以前保存したプロジェクトを開きます

③`ユーザーマニュアル`：ソフトウェア公式ウェブサイトにアクセスし、関連ガイドを確認します

④`チュートリアルと例`：クラウドスペースを通じて他のユーザーが共有したプロジェクトを表示します

⑤`最近開いたプロジェクト`：最近アクセスしたプロジェクトをすばやく開きます

⑥`シリアルツール`：シリアル通信ツール、シリアルモニター

⑦`AIツール`：AI機能を使用、Q&AモードとエージェントモードでLます

⑧`クラウドスペース`：クラウドスペースを通じてプロジェクトを保存したり、保存したプロジェクトを他のユーザーと共有したりします

⑨`ユーザーログイン`：ユーザーアカウントにログインし、AI機能とクラウドスペース機能を使用します

## 開発ボードを選択して新規プロジェクトを作成  
1. 新規プロジェクトをクリック

![](../img/docpic/doc01.png)

2. 必要な開発ボードを選択

![](../img/docpic/doc02.png)

3. 選択後、下の「このボードを使用」をクリック

![](../img/docpic/doc03.png)

4. 「プロジェクトを作成」をクリック

![](../img/docpic/doc04.png)

これで、必要な開発ボードのプロジェクトを正常に作成しました

## サンプルプログラムを選択して新規プロジェクトを作成
開発ボードを選択して作成する以外に、他の人がアップロードしたサンプルプログラムを選択することもできます。スタートバーの「チュートリアルと例」をクリックします

![](../img/docpic/doc05.png)

チュートリアルと例を開くと、他のユーザーがアップロードしたサンプルプログラムが表示されます。検索を使用して必要なサンプルを直接見つけることもできます

![](../img/docpic/doc06.png)

必要なサンプルを選択したら、サンプルを読み込むをクリックします

![](../img/docpic/doc07.png)

読み込みが完了すると、サンプルプログラムが表示され、このサンプルのライブラリファイルが自動的にインストールされます。その後、サンプルを自分で変更することもできます

![](../img/docpic/doc08.png)

## ライブラリのインストールと使用

新規プロジェクトを作成するかサンプルを読み込んだ後、左側のモジュールエリアの下部に「ライブラリ管理」が表示されます

![](../img/docpic/doc09.png)

「ライブラリ管理」をクリック

![](../img/docpic/doc10.png)

ライブラリ管理の上部のカテゴリで必要なライブラリを見つけることができます。また、必要なライブラリの名前または機能を直接検索してインストールをクリックすることもできます。

ここではDHT11温湿度センサーライブラリを検索し、インストールをクリックすると、数秒後にインストール成功のプロンプトが表示されます

![](../img/docpic/doc11.png)

## インストールされたライブラリの削除

すでにインストールされているライブラリファイルについては、引き続きライブラリ管理をクリックして入ります。インストール済みのカテゴリで、インストールされたライブラリファイルを見つけることができます。削除をクリックすると、以前にインストールされたライブラリを削除できます

![](../img/docpic/doc12.png)
