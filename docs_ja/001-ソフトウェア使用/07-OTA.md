# OTA

## BLE OTA
現在のバージョンではBLE OTA機能をサポートしています。BLE機能に対応したESP32チップに限り使用できます。
使用方法：
1. 初回使用時は、「BLE OTA」ライブラリをインストールし、プロジェクト初期化部分に「BLE OTAをワンクリックで起動」ブロックを追加してから、このプログラムをコンパイルして書き込みます。
2. 上部のデバイスポート選択メニューで「BLEデバイスを検索」オプションをクリックすると、BLE OTAに対応したデバイスが検索されます。その後、対象デバイスを選択してください。
3. アップロードボタンをクリックすると、BLE OTA方式でアップロードできます。

> **注意事項**
> 1. BLE OTAで更新する新しいファームウェアにもBLE OTA機能を含める必要があります。含まれていない場合、次回以降BLE OTAを使用できなくなります。  
> 2. ESP32のパーティション方式では、OTAパーティションがない方式を選択しないでください。
> **「BLEデバイスを検索」の表示条件**
> ユーザーがESP32の開発ボードを使用し、「BLE OTA」ライブラリをインストールしている場合、上部のデバイスポート選択メニューに「BLEデバイスを検索」オプションが表示されます。
