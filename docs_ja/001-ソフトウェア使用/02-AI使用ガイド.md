# AI機能使用ガイド
AI会話を開始する方法：

右上のAI機能をクリックすると、AI会話機能を開始できます

![](../img/aimode/ai01.png)

左下でQ&Aモジュールとエージェント（agent）モジュールを切り替えることができます

![](../img/aimode/ai02.png)

### Q&Aモードの使用

Q&Aモジュールは大規模モデルを通じて会話を行い、使用中に遭遇した質問に答えることができます
AIに直接質問する

![](../img/aimode/ai03.png)

AIが適切な解決策を提供します

![](../img/aimode/ai04.png)

### エージェント（agent）モードの使用

エージェント（agent）モードは、左下の「Q&A」を「エージェント」に切り替えるだけです

![](../img/aimode/ai05.png)

agentモードは完全に自動的にプログラムを生成できます


agentモードのAIは自動的にライブラリファイルをインストールし、グラフィックブロックを生成します
## プロジェクト分析
開発ボード、ライブラリの推奨とインストール
AIに要件を提出すると、AIが自動的にタスクを配置し、段階的に実行します
