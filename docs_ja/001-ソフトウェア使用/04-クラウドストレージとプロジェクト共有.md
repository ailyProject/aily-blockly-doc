# クラウドストレージとプロジェクト共有

## クラウドストレージ機能の使用

アカウントにログインすると、クラウドストレージ機能を使用してプロジェクトをクラウドに保存できます。コンピューターを変更した後でも、クラウドからローカルに直接ロードできます

`クラウドスペース`をクリックして自分のクラウドストレージを開きます

![](../img/cloudsave/cl01.png)

メインインターフェースから開くと、以前に同期されたプロジェクトが表示されます

プロジェクトページから開くと、`このプロジェクトをクラウドに同期`オプションが表示されます

![](../img/cloudsave/cl02.png)


`このプロジェクトをクラウドに同期`をクリックすると、プロジェクトをクラウドに保存できます

## プロジェクト共有

クラウドに保存されたプロジェクトは、他の人と共有することもできます

保存後、マウスをプロジェクト上に移動すると、4つのオプションが表示されます

`開く` `編集` `プライベート/パブリック` `削除`

![](../img/cloudsave/cl03.png)


開く：このクラウドプロジェクトを開く

編集：プロジェクトのカバー/説明/プロジェクト名などを編集

![](../img/cloudsave/cl04.png)


プライベート/パブリック：自分のプロジェクトを`チュートリアルと例`で他の人と共有するかどうかを選択します。パブリックを選択すると、`チュートリアルと例`で自分のパブリックプロジェクトを見つけることができます

![](../img/cloudsave/cl05.png)


削除：このクラウドプロジェクトを削除
