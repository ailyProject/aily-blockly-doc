# aily Project
aily Projectは、多くのAI機能を統合し、ハードウェア開発者がより快適に開発できるようにするハードウェア開発統合環境です。

## aily blockly  
aily blocklyは、aily Projectが提供する最初のIDEです。初期段階では非専門ユーザー向けにAI支援プログラミング機能を提供し、長期目標はプロフェッショナル開発と非プロフェッショナル開発の境界を打ち破り、最終的に自然言語プログラミングを実現することです。

### 現バージョンのハイライト  
1. エンジニアリングプロジェクト管理  
npmを使用してプロジェクト管理を行い、プロジェクト単位で開発ボードとライブラリを管理します。従来の組み込み開発環境のエンジニアリング不足の問題を多く解決しました。例えば、Arduino IDEを使用するとボードパッケージ、ライブラリ、現在のプロジェクトが一致せず、コンパイル失敗や実行エラーが発生する可能性があります。本ソフトウェアでは、各プロジェクトの開発ボードバージョンとライブラリバージョンは独立しており、プロジェクト間で影響を受けません。
2. ライブラリマネージャー  
多くのライブラリを準備しましたが（ほぼすべての一般的なモジュールをカバー）、実際にはこれらのライブラリはAIによって生成されており、詳細な検証は行われていません。ベータテスト参加者と一緒に検証と改善を行う必要があります。
3. 万能でコンパクトなシリアルデバッグツール  
万能なシリアルツールを作ろうとしています。テスト、フィードバック、新しいアイデアの提案をお待ちしています。
4. AIプロジェクト生成  
ユーザーのニーズに基づいて自動的にプロジェクトを分析し、開発ボード、モジュール、ライブラリを推奨し、プロジェクトアーキテクチャ図、ピン接続図を生成し、ユーザーのためにプロジェクトを作成します。
5. AIコード生成  
ユーザーのニーズに基づいて自動的にプログラムを作成します
6. AIライブラリ変換  
ネイティブC/C++ライブラリは、本ソフトウェアで使用するライブラリに簡単に変換できます。大規模モデルに基づく設定生成により、開発中にArduinoライブラリを使用したいが対応するblocklyライブラリがない場合、ArduinoライブラリをAIに提供するだけで、AIが自動的に分析し、対応するblocklyライブラリを生成します。この機能により、本ソフトウェアは最も多くのblocklyを持つ開発プラットフォームになることができます。
7. AI開発ボード設定生成  
大規模モデルに基づく設定生成により、開発ボードを追加する際に手動で新しい設定を書く必要はありません。開発ボードのドキュメント（md形式）を提供するだけで、AIが自動的に分析し、開発ボード設定ファイルを生成します。（esp32、avr、renesas、rp2040、stm32をコアとする開発ボードのみサポート。コンパイラとコアSDKは事前にリポジトリに準備する必要があります）
8. ライトニングコンパイルツール（フェーズ1はオンライン、フェーズ2はさらに高速化！）  
エッジクラウド連携、ライトニングコンパイル  
元々1時間かかっていたコンパイル作業を1分に短縮！

## 非正式バージョンに関する注意事項  
今回テストするアルファ版は、最低限の機能のみを保証しており、計画されている多くのハイライト機能はまだ設計・開発が完了していません。
現在のバージョンは実際の作業には推奨されません。後で多くの調整を行う可能性があり、バージョン間の非互換性が生じる可能性があります。

## 使用している主なオープンソースプロジェクト
[electron]()本プロジェクトはelectronを使用してデスクトップアプリケーションを構築しています  
[angular]()本プロジェクトはangularをレンダリングエンジンとして使用し、主要なUIロジックを構築しています  
[node]()本プロジェクトはnpmとnodeを使用してパッケージ管理と必要なスクリプトの実行を行っています  
[7z]()本プロジェクトは7zを使用して一部のパッケージのサイズを縮小しています（巨大なESP32コンパイラなど）  
その他の内容は[package.json](https://github.com/ailyProject/aily-blockly/blob/master/package.json)を参照してください
