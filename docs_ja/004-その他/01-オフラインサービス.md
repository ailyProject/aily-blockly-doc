# aily blocklyオフラインサービス  
`本プログラムは教育目的のみに使用でき、商業目的での使用は禁止されています`  
aily blocklyオフラインサービスは、aily blocklyのすべてのリソースをローカルコンピューターにキャッシュし、ユーザーがオフライン環境でaily blockly関連の開発ボード、ライブラリなどのリソースを使用できるようにします。

## クイックスタート
0. [nodejs](https://npmmirror.com/mirrors/node/v22.21.1/node-v22.21.1-x64.msi)をダウンロードしてインストール。  
1. [aily blocklyオフラインサービスプログラム](https://github.com/ailyProject/aily-blockly-offline-service/archive/refs/heads/main.zip)をダウンロードして解凍
2. 解凍したディレクトリに入り、空白部分を右クリックして`ターミナルで開く`を選択し、以下のコマンドを実行して`関連する依存関係をインストール`：
```shell
npm i
```
> 本サービスを初めて使用する場合のみ、依存関係をインストールする必要があります  

3. 以下のコマンドを使用して`オフラインサービスを実行`：  
```shell
node cli.js run
```
> 再起動後はこのコマンドを使用してオフラインサービスを有効にできます。  

4. 以下のコマンドを実行して`ローカルnpmリポジトリと関連リソースを更新`：  
```shell
node cli.js update
```
> npmリポジトリを更新する必要があるたびにこのコマンドを使用できます

5. aily blocklyソフトウェアを開き、左上のメニューから設定ポップアップを開き、リポジトリ設定をクリックしてローカルリポジトリを選択。

これで設定が完了し、ソフトウェアで新しいプロジェクトを作成したり、ライブラリをインストールしたりするときにローカルサービスを使用します。

## サービスの停止
以下のコマンドを実行してサービスを停止できます
```shell
node cli.js stop
```
aily blocklyの設定で、リポジトリ設定を変更して公式クラウドサービスを使用できます
