# 开发者模式  
如果您正在适配库、开发板，非常建议您打开开发者模式。

## 打开开发者模式
通过`菜单 > 设置 > 开发者模式`，打开开发者模式。
此时blockly工作区左下角的会出现`开发者悬浮菜单`。  

![](../img/developer/dev-tool.webp)


## 开发者功能
当前提供的开发者功能有：  
1. 重载项目。  
  点击后可以重新加载整个项目。如果开发者修改了库的blockly部分，点击可以重载修改后的库；  
2. 清除缓存。  
  点击后可以清除编译缓存。如果开发修改了库src源码，且之前编译过，那需要清除一下缓存，避免系统使用之前的编译缓存； 
3. 打开全局资源文件夹
  点击后会通过系统文件管理器，打开
4. 打开编译文件夹

5. 自动保存
  默认勾选，勾选后在重载项目前，会进行先保存项目。

## 其他
后续开发者需要的功能，我们也会添加到这里。  
悬浮菜单可拖动到窗口其他位置显示，避免遮挡用户操作。
