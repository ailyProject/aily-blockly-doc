# AI功能使用指南
开启AI对话方法：

点击右上角AI功能，即可开始AI对话功能

![](..\img\aimode\ai01.png)

左下角可以在问答模块与代理（agent）模块之间进行切换

![](..\img\aimode\ai02.png)

### 问答模式使用

问答模块可以通过大模型进行对话，回答我们使用过程中遇到的问题
直接向AI提问

![](..\img\aimode\ai03.png)

AI会给出你合适的方案

![](..\img\aimode\ai04.png)

### 代理（agent）模式使用

代理（agent）模式只需要在左下角将"问答"切换为"代理"

![](..\img\aimode\ai05.png)

agent模式可以全自动的帮你生成程序


agent模式下的AI会自动帮你安装库文件，生成图形块
## 项目分析
开发板、库的推荐与安装
对AI提出你的要求，AI会自动安排任务并逐步执行
