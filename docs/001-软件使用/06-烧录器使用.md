# 烧录器使用
软件中集成有openocd/probe-rs，支持使用dapLink、J-Link、ST-Link烧录器。
其中daplink直接可以使用，J-Link、ST-Link需要使用Zadig替换驱动后才能使用。  


### J-Link/ST-Link替换驱动方法
0. 下载[Zadig](https://zadig.akeo.ie/)。
1. `管理员权限`打开 Zadig。
2. 点击菜单栏 Options -> List All Devices。
3. 在下拉列表中，你会看到几个和 J-Link/ST-Link 相关的条目。请仔细甄别：  

J-Link注意事项： 
> 选择叫 J-Link (Interface 0) 的设备  
> 确认其USB ID：通常VID为1366  

ST-Link注意事项：
> 选择叫 ST-Link Debug(Interface 0) 的设备  
> 确认其USB ID：通常VID为0483

4. 在右侧驱动选择框中选择 WinUSB，点击 Replace Driver。  
5. 等待替换完成。

现在再打开本软件，即可看到对应设备，并使用其烧录芯片/开发板了。

### 还原J-Link驱动
一旦替换为 WinUSB，Segger 官方的软件（如 J-Flash、J-Link Configurator）将无法识别该设备。如果你想换回官方工具，需要在设备管理器里卸载设备并勾选“删除驱动”，然后重插设备，则可使用原版驱动。

### 已知问题
一些新型的STM32无法正常烧录，如STM32F407，可能和调试器本身固件也有关系。
