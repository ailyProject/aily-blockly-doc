# 软件安装  

## 下载软件
下载地址1（中国大陆）：https://arduino.me/a/aily-blockly  
下载地址2（海外地区）：https://github.com/ailyProject/aily-blockly/releases  

## windows安装
从下载链接中下载好当前版本的软件

![](../img/install/win01.png)

双击安装程序，可以点击浏览，更改aily-blockly的安装目录

![](../img/install/win02.png)

等待安装完成

![](../img/install/win03.png)

点击`完成`，结束此次安装

![](../img/install/win04.png)

## Mac安装
1. 双击安装程序(如:aily-blockly-0.8.8.dmg)，便会出现安装界面：
![](../img/install/mac_1.png)
    将`aily blockly`图标拖往`Applications`中


2. 从`启动台`中便可找到 aily blockly应用
![](../img/install/mac_3.png)


3. 初次打开会弹出如下确认信息，点击打开即可
![](../img/install/mac_2.png)
