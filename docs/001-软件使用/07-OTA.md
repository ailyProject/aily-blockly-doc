# OTA

## BLE OTA
当前版本已支持BLE OTA功能，仅限支持BLE功能ESP32芯片。
使用方法如下：
1. 首次使用，需安装“BLE OTA”库，然后向项目初始化部分，添加“一键启动BLE OTA”块，然后编译烧录本程序。 
2. 在顶部的设备端口选择菜单中，点击“搜索BLE设备”选项，可会搜索支持BLE OTA的设备，然后选择指定设备即可。
3. 现在点击上传按钮，则可使用BLE OTA方式进行上传了。

> **注意事项**
> 通过BLE OTA更新的新固件，也应该具备BLE OTA功能，否则会导致后续再次使用BLE OTA失败  
> **"搜索BLE设备"显示条件**
> 当用户使用esp32的开发板，且安装了“BLE OTA”库后，即可在顶部的设备端口选择菜单中看到“搜索BLE设备”选项
