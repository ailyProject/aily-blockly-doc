# aily blockly离线服务  
`本程序仅供教育用途，禁止用于商业用途`  
aily blockly离线服务，可以将aily blockly所有资源缓存到本地计算机，使得用户可以在离线环境下使用aily blockly相关开发板、库等资源。

## 快速使用
0. 下载[nodejs](https://npmmirror.com/mirrors/node/v22.21.1/node-v22.21.1-x64.msi)，并安装。  
1. 下载[aily blockly离线服务程序](https://github.com/ailyProject/aily-blockly-offline-service/archive/refs/heads/main.zip)，并解压
2. 进入解压后的目录，右键点击空白位置，选择`在终端中打开`，运行以下指令`安装相关依赖`：
```shell
npm i
```
> 只有第一次使用本服务，才需要安装依赖  

3. 使用如下指令，`运行离线服务`：  
```shell
node cli.js run
```
> 此后每次重新开机后，都可以使用该指令启用离线服务。  

4. 运行以下命令，`更新本地npm仓库`：  
```shell
node cli.js update
```
> 此后每次需要更新npm仓库时，都可以使用该命令
    
5. 运行以下命令，`更新相关资源`:  
```shell
node cli.js sync
```
> 此后需要更新资源时，都可以使用该命令  

6. 打开aily blockly软件，通过左上菜单，打开设置弹窗，点击仓库设置，选择本地仓库。

现在已经配置完成，你在软件中新建项目、安装库都将使用本地服务了。

## 停止服务
运行以下指令可停止服务
```shell
node cli.js stop
```
在aily blockly设置中，可以修改仓库设置，使用官方云服务
