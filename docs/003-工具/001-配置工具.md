# aily Config Tool 
本程序用于手机、PC配置aily HDK使用。
项目地址：[aily Config Tool](https://github.com/ailyProject/aily-config-tool)

主要功能如下:
1. 系统基础信息查看
2. 配置WiFi
3. 配置模型
4. 查看对话记录


## 适配硬件
· Raspberry Pi zero W/2W  [已集成aily服务的镜像](https://github.com/ailyProject/aily-config-service)  


## 问题反馈
