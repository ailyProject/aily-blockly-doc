# 燒錄器使用
軟體中整合有openocd/probe-rs，支援使用dapLink、J-Link、ST-Link燒錄器。
其中daplink可直接使用，J-Link、ST-Link需要使用Zadig替換驅動後才能使用。  


### J-Link/ST-Link替換驅動方法
0. 下載[Zadig](https://zadig.akeo.ie/)。
1. 以`管理員權限`開啟 Zadig。
2. 點選選單列 Options -> List All Devices。
3. 在下拉列表中，你會看到幾個和 J-Link/ST-Link 相關的條目。請仔細甄別：  

J-Link注意事項： 
> 選擇名為 J-Link (Interface 0) 的裝置  
> 確認其USB ID：通常VID為1366  

ST-Link注意事項：
> 選擇名為 ST-Link Debug(Interface 0) 的裝置  
> 確認其USB ID：通常VID為0483

4. 在右側驅動選擇框中選擇 WinUSB，點選 Replace Driver。  
5. 等待替換完成。

現在再開啟本軟體，即可看到對應裝置，並使用其燒錄晶片/開發板了。

### 還原J-Link驅動
一旦替換為 WinUSB，Segger 官方的軟體（如 J-Flash、J-Link Configurator）將無法識別該裝置。如果你想換回官方工具，需要在裝置管理員裡解除安裝裝置並勾選「刪除驅動」，然後重新插入裝置，即可使用原版驅動。

### 已知問題
一些新型的STM32無法正常燒錄，如STM32F407，可能和除錯器本身韌體也有關係。
