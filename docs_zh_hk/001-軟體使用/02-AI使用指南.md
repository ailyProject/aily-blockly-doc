# AI功能使用指南
開啟AI對話方法：

點擊右上角AI功能，即可開始AI對話功能

![](../img/aimode/ai01.png)

左下角可以在問答模組與代理（agent）模組之間進行切換

![](../img/aimode/ai02.png)

### 問答模式使用

問答模組可以透過大模型進行對話，回答我們使用過程中遇到的問題
直接向AI提問

![](../img/aimode/ai03.png)

AI會給出你合適的方案

![](../img/aimode/ai04.png)

### 代理（agent）模式使用

代理（agent）模式只需要在左下角將「問答」切換為「代理」

![](../img/aimode/ai05.png)

agent模式可以全自動的幫你生成程式


agent模式下的AI會自動幫你安裝庫檔案，生成圖形塊
## 專案分析
開發板、庫的推薦與安裝
對AI提出你的要求，AI會自動安排任務並逐步執行

![](../img/aimode/ai06.png)

## 程式碼生成

![](../img/aimode/ai07.png)
