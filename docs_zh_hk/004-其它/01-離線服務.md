# aily blockly離線服務  
`本程式僅供教育用途，禁止用於商業用途`  
aily blockly離線服務，可以將aily blockly所有資源快取到本機電腦，使得用戶可以在離線環境下使用aily blockly相關開發板、庫等資源。

## 快速使用
0. 下載[nodejs](https://npmmirror.com/mirrors/node/v22.21.1/node-v22.21.1-x64.msi)，並安裝。  
1. 下載[aily blockly離線服務程式](https://github.com/ailyProject/aily-blockly-offline-service/archive/refs/heads/main.zip)，並解壓縮
2. 進入解壓縮後的目錄，右鍵點擊空白位置，選擇`在終端中開啟`，執行以下指令`安裝相關依賴`：
```shell
npm i
```
> 只有第一次使用本服務，才需要安裝依賴  

3. 使用如下指令，`執行離線服務`：  
```shell
node cli.js run
```
> 此後每次重新開機後，都可以使用該指令啟用離線服務。  

4. 執行以下命令，`更新本機npm倉庫以及相關資源`：  
```shell
node cli.js update
```
> 此後每次需要更新npm倉庫時，都可以使用該命令

5. 打開aily blockly軟體，透過左上選單，打開`設定`視窗，點擊`服務區域`，選擇`Local`。

現在已經配置完成，你在軟體中新建專案、安裝庫都將使用本機服務了。

## 停止服務
執行以下指令可停止服務
```shell
node cli.js stop
```
在aily blockly設定中，可以修改倉庫設定，使用官方雲端服務
