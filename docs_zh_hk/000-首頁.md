# aily Project
aily Project是一個硬體開發整合環境，計劃整合諸多AI能力，幫助硬體開發者更暢快地進行開發。

## aily blockly  
aily blockly是aily Project推出的首款IDE，前期面向非專業用戶提供AI輔助編程能力，長遠目標是打破專業開發和非專業開發的界限，最終實現自然語言編程。

### 當前版本亮點  
1. 工程化專案管理  
使用npm進行專案管理，做到以專案為單位進行開發板和庫的管理。解決了諸多傳統嵌入式開發環境的工程化不足的問題。如，使用Arduino IDE可能出現board package、庫和當前專案不匹配，造成編譯失敗，運行錯誤的問題。在本軟體上，各專案中的開發板版本和庫版本是獨立的，專案間互不影響。
2. 庫管理器  
雖然我們已經準備了很多庫（幾乎涵蓋了常用模組），但實際上這些庫都是AI生成的，我們沒有經過詳細驗證。需要內測參與者和我們一道進行驗證和完善。
3. 全能且小巧的串口調試工具  
試圖打造一個全能的串口工具，歡迎大家測試、反饋、提出新的想法。
4. AI專案生成  
根據用戶需求，自動分析專案，推薦開發板、模組、庫，生成專案架構圖、引腳連接圖，並為用戶創建出專案。
5. AI程式碼生成  
根據用戶需求，自動編寫程式
6. AI轉庫  
原生C/C++庫都可以輕鬆轉換成本軟體使用的庫。基於大模型的配置生成，開發過程中，如果想使用arduino庫，但沒有對應的blockly庫，只用將arduino庫提供給AI，AI自動分析，生成對應的blockly庫。藉助該功能，本軟體可以成為blockly最多的開發平台。
7. AI開發板配置生成  
基於大模型的配置生成，添加開發板時不用再純手寫新配置，只用提供開發板文檔（md格式），AI自動分析，幫你生成開發板配置檔案。（僅支持esp32、avr、renesas、rp2040、stm32為核心的開發板，因為編譯器和核心sdk，還是需要我們提前準備到倉庫的）
8. 閃電編譯工具（一期已上線，二期還將提速！）  
端雲協同，閃電編譯  
將原本1小時的編譯工作縮短到1分鐘！

## 非正式版注意事項  
本次測試的alpha版本，僅保證最低限度的能用，很多計劃的亮點功能還未完成設計和開發。
當前版本不建議實際用於工作，因為後期我們做出的諸多調整，可能會導致版本間的不兼容。

## 計劃功能
· AI加持（專案模板生成、塊/庫生成、自動調試）    
· 多版本開發板、庫共存管理  
· 硬體仿真

## 專案使用到的主要開源專案
[electron]()本專案使用electron建構桌面程式
