# 開發者模式  
如果您正在適配庫、開發板，非常建議您打開開發者模式。

## 打開開發者模式
透過`選單 > 設定 > 開發者模式`，打開開發者模式。
此時blockly工作區左下角的會出現`開發者懸浮選單`。  

![](../img/developer/dev-tool.webp)


## 開發者功能
當前提供的開發者功能有：  
1. 重新載入專案。  
  點擊後可以重新載入整個專案。如果開發者修改了庫的blockly部分，點擊可以重新載入修改後的庫；  
2. 清除快取。  
  點擊後可以清除編譯快取。如果開發修改了庫src原始碼，且之前編譯過，那需要清除一下快取，避免系統使用之前的編譯快取； 
3. 打開全域資源資料夾
  點擊後會透過系統檔案管理器，打開
4. 打開編譯資料夾

5. 自動儲存
  預設勾選，勾選後在重新載入專案前，會進行先儲存專案。

## 其他
後續開發者需要的功能，我們也會添加到這裡。  
懸浮選單可拖動到視窗其他位置顯示，避免遮擋用戶操作。
