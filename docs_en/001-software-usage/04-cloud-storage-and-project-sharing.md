# Cloud Storage and Project Sharing

## Cloud Storage Function Usage

After logging in, you can store projects in the cloud through the cloud storage function. After changing computers, you can also directly load them from the cloud to local

Click `Cloud Space` to open your cloud storage

![](../img/cloudsave/cl01.png)

When opened from the main interface, it will display previously synced projects

When opened from your project page, the option `Sync this project to cloud` will appear

![](../img/cloudsave/cl02.png)


Click `Sync this project to cloud` to store the project in the cloud

## Project Sharing

Projects stored in the cloud can also be shared with others

After storing, move the mouse to the project, and you can see four options

`Open` `Edit` `Private/Public` `Delete`

![](../img/cloudsave/cl03.png)


Open: Open this cloud project

Edit: Edit the project's cover/introduction/project name, etc.

![](../img/cloudsave/cl04.png)


Private/Public: Choose whether to make your project public to `Tutorials & Examples` to share with others. If you choose public, you can find your public project in `Tutorials & Examples`

![](../img/cloudsave/cl05.png)


Delete: Delete this cloud project
