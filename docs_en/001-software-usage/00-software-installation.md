# Software Installation  

## Download Software
Download address 1: https://aily.pro/download  
Download address 2 (Github): https://github.com/ailyProject/aily-blockly/releases  

## Windows Installation
Download the current version of the software from the download link

<img src="../img/install/win01.png" width="100">

Double-click the installer, you can click Browse to change the installation directory of aily-blockly

<img src="../img/install/win02.png" width="600">

Wait for installation to complete

<img src="../img/install/win03.png" width="600">

Click `Finish` to end the installation

<img src="../img/install/win04.png" width="600">

## Mac Installation
1. Double-click the installer (e.g., aily-blockly-0.8.8.dmg), and the installation interface will appear:
![](../img/install/mac_1.png)  

Drag the `aily blockly` icon to `Applications`


2. You can find the aily blockly application from `Launchpad`  
<img src="../img/install/mac_3.png" width="200">


3. When opening for the first time, the following confirmation message will pop up, click Open  
<img src="../img/install/mac_2.png" width="300">



## Run the Program

Double-click the aily blockly icon to open the software, and after a moment you will see the main interface  

<img src="../img/install/win05.png" width="600">
