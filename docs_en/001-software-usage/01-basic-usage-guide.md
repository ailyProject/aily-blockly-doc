# Basic Usage Guide  

## Guide Page Features
![](../img/main/main01.png)

① `New Project`: Create a new project

② `Open Project`: Open a previously saved project

③ `User Manual`: Visit the software official website to view relevant guides

④ `Tutorials & Examples`: View projects shared by other users through cloud space

⑤ `Recent Projects`: Quickly open projects you recently accessed

⑥ `Serial Tool`: Serial communication tool, serial monitor

⑦ `AI Tools`: Use AI features, available in Q&A mode and agent mode

⑧ `Cloud Space`: Save your projects through cloud space, or share your saved projects with other users

⑨ `User Login`: Log in to your user account to use AI features and cloud space features

## Create New Project by Selecting Development Board  
1. Click New Project

![](../img/docpic/doc01.png)

2. Select the development board you need

![](../img/docpic/doc02.png)

3. After selection, click "Use This Board" below

![](../img/docpic/doc03.png)

4. Click "Create Project"

![](../img/docpic/doc04.png)

Now, we have successfully created a project for our desired development board

## Create New Project by Selecting Example Program
In addition to creating by selecting a development board, we can also select example programs uploaded by others, click "Tutorials & Examples" in the start menu

![](../img/docpic/doc05.png)

After opening Tutorials & Examples, you will see example programs uploaded by other users, or you can search directly to find the relevant examples you need

![](../img/docpic/doc06.png)

After selecting the example you need, click Load Example

![](../img/docpic/doc07.png)

After loading is complete, you can see the example program, and the library files for this example will be automatically installed, and you can also modify the example later

![](../img/docpic/doc08.png)

## Install and Use Libraries

After creating a new project or loading an example, you can see "Library Manager" at the bottom of the module area on the left

![](../img/docpic/doc09.png)

Click "Library Manager"

![](../img/docpic/doc10.png)

You can find the library you need in the categories at the top of the library manager, or you can directly search for the name or function of the library you need, and click Install.

Here we search for the DHT11 temperature and humidity sensor library, click Install, and after a few seconds it will prompt that the installation was successful

![](../img/docpic/doc11.png)

## Remove Installed Library

For libraries that have already been installed, we still click to enter the library manager. Under the Installed section, you can find the installed library files. Click Remove to remove the previously installed library

![](../img/docpic/doc12.png)
